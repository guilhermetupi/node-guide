# Running the project

## Clone this repo
$ git clone <https://github.com/guilhermetupi/node-guide.git>

## Install dependencies
With npm:
```$ npm i```

With yarn:
```$ yarn add```

## Run the project
With npm:
```$ npm start```

With yarn:
```$ yarn start```
